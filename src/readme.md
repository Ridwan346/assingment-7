1/JSX (JavaScript XML) হলো JavaScript–এর একটি syntax extension
এতে আমরা HTML এর মতো কোড JavaScript এর ভেতরে লিখতে পারি।

2/State এবং Props এর পার্থক্য
Props = বাইরে থেকে আসে
State = ভিতরের ডাটা

3/useState হলো একটি React Hook
এটি দিয়ে আমরা component এর ভেতরে state তৈরি ও পরিবর্তন করতে পারি

4/React এ component এর মধ্যে State শেয়ার করা
সবচেয়ে সাধারণ উপায়: Lifting State Up

5/React এ event লেখা হয় camelCase এ
এবং function হিসেবে দেওয়া হয়।
